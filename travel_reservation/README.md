# Luxury Travel Reservation System

A full-stack luxury travel reservation website built with Flask, Tailwind CSS, and Three.js.

## Features

- **User Authentication**: Secure login and registration system
- **Destination Management**: Browse and search luxury destinations
- **Booking System**: Make and manage travel bookings
- **Admin Panel**: Manage destinations and view bookings
- **Responsive Design**: Works on all device sizes
- **Modern UI**: Glassmorphism design with smooth animations
- **3D Effects**: Three.js particle system in the hero section

## Tech Stack

- **Backend**: Flask, SQLAlchemy
- **Frontend**: HTML, Tailwind CSS, JavaScript
- **Database**: SQLite
- **3D Graphics**: Three.js
